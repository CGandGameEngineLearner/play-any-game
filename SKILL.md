# Play Any Game - AI游戏伴侣

## 核心能力

让AI agent成为你的游戏伴侣，帮你处理游戏中的重复性任务。

## 工作模式

```
你设定目标 → AI理解任务 → 执行并反馈 → 完成汇报
```

## 功能列表

### 1. 日常任务代管
- 每日委托/任务自动完成
- 体力/理智智能消耗
- 活动副本自动刷取

### 2. 副本刷取辅助
- 指定副本循环刷取
- 刷取次数和目标设定
- 奖励自动记录统计

### 3. 进度管理
- 资源获取记录
- Box数据整理分析
- 养成进度追踪

### 4. 游戏顾问
- 配队建议
- 养成优先级
- 攻略咨询

## 使用示例

### 崩坏星穹铁道

```
用户：帮我把今天的开拓力清完
AI：开始执行...
- 检测当前开拓力：180/240
- 进入历战余响...
- 本次获得：角色突破材料×2
- 继续下一个副本...
完成！共消耗160点开拓力，获得星琼×40，尘土×15
```

### 原神

```
用户：帮我刷20次绝缘套
AI：好的，开始刷取...
- 进入副本：炽烈的炎之魔女
- 等待战斗结束...
- 领取奖励...
- 重新挑战（剩余19次）...
完成！共刷取20次，有效圣遗物3个，消耗树脂160
```

## 技术实现

使用 `game-ai-controller` skill 进行游戏UI扫描和操作控制：
- 屏幕截图识别
- UI元素定位
- 点击/滑动操作
- 状态判断

## 注意事项

- 需要游戏窗口保持在可见状态
- 部分游戏可能有检测机制，请自行评估风险
- 长时间运行时注意设备状态
- 合理安排游戏时间
