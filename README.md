# 🎮 Play Any Game - AI游戏向导

> **你的专属游戏向导！无论是刚入坑的新手还是老玩家，AI都能陪你一起探索游戏世界，解答疑惑、规划成长路线，让游戏体验更加顺畅有趣！**

## ✨ 为什么需要AI游戏向导？

- 😔 **刚入坑一脸懵？** - AI帮你快速了解游戏机制，少走弯路
- 🤔 **卡关不知道怎么办？** - AI给你详细的攻略和配队建议
- 📊 **资源不知道怎么分配？** - AI帮你规划最优养成路线
- 🎮 **一个人玩游戏太无聊？** - AI随时陪你聊游戏、讨论攻略
- ⏰ **没时间研究攻略？** - 告诉AI你的目标，它帮你制定计划

## 🚀 能帮你的场景

| 场景 | 示例 |
|------|------|
| 新手入门 | "这个游戏怎么玩？" |
| 卡关求助 | "这个Boss打不过怎么办" |
| 配队咨询 | "帮我配个队伍" |
| 资源规划 | "我应该先练哪个角色？" |
| 攻略查询 | "活动副本怎么打？" |
| 日常建议 | "今天该做什么？" |

## 💬 就像有个老玩家在身边

```markdown
你：这套圣遗物哪个好？
AI：让我看看...根据你的角色和当前副本，我认为2号套更合适。
     它的主词条是暴击率，能弥补你队伍的短板。配上这套输出更稳定~

你：今天没太多时间玩
AI：没关系！今天建议优先：1）清日常（10分钟）
     2）把体力刷完（15分钟）这样基本不会落下进度~
```
