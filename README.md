# 🎮 Play Any Game - AI游戏伴侣

> **厌倦了游戏里日复一日的重复劳动？让AI成为你的游戏伴侣，帮你处理那些繁琐的日常任务，你可以把时间花在真正有意思的内容上！**

## ✨ 这个技能能做什么？

当你有事走开或者懒得动手的时候，可以让AI来陪你"玩"：

- 🎮 **AI代管日常** - 委托任务、体力消耗、副本次数...AI帮你盯着
- 📊 **智能进度管理** - 自动记录你的资源获取情况，整理box数据
- 💡 **游戏攻略建议** - 根据你的情况给出最优策略
- ⏰ **省心省力** - 设定好目标，AI帮你一步步完成

## 🚀 快速体验

```markdown
用户：AI帮我把今天的日常和体力清一下
AI：好的！我来帮你处理...
- 进入委托任务...
- 消耗开拓力刷取...
- 检查是否有遗漏...
完成！今日任务已全部清理，获得星琼×80，尘土×20
```

## 📋 适用场景

| 场景 | 示例 |
|------|------|
| 日常代管 | "帮我清一下今日体力" |
| 副本规划 | "帮我刷20次这个副本" |
| 进度追踪 | "我本周刷了哪些资源？" |
| 攻略咨询 | "忘却之庭10层怎么过？" |

## 🔧 技术说明

本技能基于 OpenClaw agent 框架，结合图像识别技术辅助AI理解游戏界面状态，提供智能化的游戏辅助服务。

**使用须知：**
- 请合理安排游戏时间，健康游戏
- 部分游戏可能不支持，请自行评估
- 本技能仅供个人学习研究使用
